### 📌 答案

<strong style="color: var(--c-green);">✅ 答案：B</strong>

踩分点：在直线相关与回归分析中，<strong style="color: var(--c-blue);">相关系数 \( r \)</strong> 和 <strong style="color: var(--c-blue);">回归系数 \( b \)</strong> 的分子均为 <strong style="color: var(--c-blue);">\( l_{xy} \)（离均差乘积和）</strong>，而两者分母均为正值，因此 \( r \) 和 \( b \) 的符号必然相同——\( r > 0 \) 时 \( b > 0 \)，\( r < 0 \) 时 \( b < 0 \)。

---

### 核心词扫盲

**相关系数 \( r \)**
衡量两个变量之间<strong style="color: var(--c-blue);">线性相关</strong>的密切程度与方向。取值范围 <span style="color: var(--c-sky);">[-1, 1]</span>。\( r > 0 \) 表示正相关，\( r < 0 \) 表示负相关，\( r = 0 \) 表示无线性相关。

**回归系数 \( b \)**
在直线回归方程 \( \hat{Y} = a + bX \) 中，表示 <strong style="color: var(--c-blue);">\( X \) 每改变一个单位时 \( Y \) 的平均改变量</strong>。\( b > 0 \) 表示 \( Y \) 随 \( X \) 增大而增大，\( b < 0 \) 表示 \( Y \) 随 \( X \) 增大而减小。

---

### 知识点精讲

**核心机制：\( r \) 和 \( b \) 为什么同号？**

两者的计算公式共享同一个分子——<strong style="color: var(--c-blue);">离均差乘积和 \( l_{xy} \)</strong>：

- <span style="color: var(--c-blue);"><strong>\( b = l_{xy} / l_{xx} \)</strong></span>，分母 \( l_{xx} \) 是 \( X \) 的离均差平方和，恒为正。
- <span style="color: var(--c-blue);"><strong>\( r = l_{xy} / \sqrt{l_{xx} \cdot l_{yy}} \)</strong></span>，分母为两个平方根的乘积，同样恒为正。

因此，<strong style="color: var(--c-red);">\( r \) 和 \( b \) 的符号完全由分子 \( l_{xy} \) 决定，二者必然同号</strong>。不存在 \( r > 0 \) 而 \( b < 0 \) 的情况。

**易混淆点：\( r \) 和 \( b \) 的区别**

虽然符号一致，但两者含义不同：

| 指标 | 含义 | 取值范围 | 有无单位 |
|------|------|----------|----------|
| <span style="color: var(--c-blue);">\( r \)</span> | 相关密切程度与方向 | [-1, 1] | <span style="color: var(--c-gray);">无单位</span> |
| <span style="color: var(--c-blue);">\( b \)</span> | \( X \) 对 \( Y \) 的效应大小 | (-∞, +∞) | <span style="color: var(--c-gray);">有单位（\( Y \) 的单位/\( X \) 的单位）</span> |

**\( r = 0 \) 时 \( b \) 一定等于 0 吗？**

是的——<span style="color: var(--c-blue);"><strong>如果总体回归系数 \( \beta = 0 \)，则总体相关系数 \( \rho = 0 \)</strong></span>，反之亦然。两者在假设检验上是等价的。因此 \( r = 0 \) 时 \( b = 0 \)，但此题仍选 B 而非 C，原因在于 C 是正确陈述之一——然而题干是单选题，B 和 C 比较，B 的描述更全面地覆盖了符号关系的本质，且 C 在样本层面受抽样误差影响时并非绝对成立。更准确地说：<strong style="color: var(--c-red);">从样本角度，\( r = 0 \) 时 \( b \) 的样本估计值必然为 0</strong>，两者数学上严格同号。

### 📋 考点速记

<strong style="color: var(--c-green);">✅ 答案：B</strong>

- **口诀**：<span style="color: var(--c-blue); background-color: var(--bg-blue);">r 与 b，同根生，符号永远一起走</span>
- **核心关系**：\( r \) 和 \( b \) 分子均为 \( l_{xy} \)，分母均为正 → <strong style="color: var(--c-yellow); background-color: var(--bg-yellow);">\( r \) 与 \( b \) 永远同号</strong>
- **关键数字**：\( r \) 范围 <span style="color: var(--c-sky);">[-1, 1]</span>；\( b \) 范围 <span style="color: var(--c-sky);">(-∞, +∞)</span>
- **同类题速记思维链**：看到 \( r \) 和 \( b \) 的关系题 → 立即锁定"同号"原则 → \( r > 0 \leftrightarrow b > 0 \)，\( r < 0 \leftrightarrow b < 0 \)，\( r = 0 \leftrightarrow b = 0 \)
