### 📌 答案

<strong style="color: var(--c-green);">✅ 答案：B</strong>

<span style="color: var(--c-blue);">一元线性回归中，回归系数 b 的符号恒等于相关系数 r 的符号</span>，因为 b = r × (s<sub>y</sub> / s<sub>x</sub>)，而标准差比值必然为正。

### 核心词扫盲

- <strong style="color: var(--c-blue);">相关系数 r</strong>：衡量两个变量间<span style="color: var(--c-blue);">线性相关方向与强度</span>的指标，取值范围 -1 ~ +1。r > 0 表示正相关，r < 0 表示负相关，r = 0 表示无线性相关。
- <strong style="color: var(--c-blue);">回归系数 b</strong>：即一元线性回归方程 Ŷ = a + bX 中的<span style="color: var(--c-blue);">斜率</span>，表示 X 每变动一个单位时 Y 的平均改变量。

### 知识点精讲

<span style="color: var(--c-indigo);">r 与 b 的符号绑定关系</span>是一元线性回归中必考的底层逻辑。

在标准的一元回归中，回归系数 b 可由公式直接表达：

<strong style="color: var(--c-blue);">b = r × (s<sub>y</sub> / s<sub>x</sub>)</strong>

其中 s<sub>y</sub> 和 s<sub>x</sub> 分别为 Y 和 X 的标准差，它们<strong style="color: var(--c-sky);">均为非负数</strong>。因此 b 的正负号 <span style="color: var(--c-red);">完全由 r 决定</span>：r 大于 0，b 必大于 0；r 小于 0，b 必小于 0；r = 0，b = 0。两者符号绑定，绝不分离。

此外，在对同一组数据的假设检验中，<span style="color: var(--c-blue);">r 的显著性检验与 b 的显著性检验是等价的</span>——检验结论永远一致。这是医学统计学中一道反复出现的考点。

### 📋 考点速记

<strong style="color: var(--c-green);">✅ 答案：B</strong>

- <span style="color: var(--c-yellow); background-color: var(--bg-yellow);">口诀：r 正 b 正，r 零 b 零，符号永远跟着走。</span>
- <span style="color: var(--c-blue);">核心公式：b = r × (s<sub>y</sub> / s<sub>x</sub>)，因标准差比恒正，故 r 与 b 永远同号。</span>
- 假设检验等价：同一组数据的 r 检验与 b 检验，结论完全一致。
- 关键区分：决定系数 r² 恒为非负，不能反映方向；符号信息必须从 r 或 b 本身读取。
