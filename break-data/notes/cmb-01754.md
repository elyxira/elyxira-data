# 📌 答案

<strong style="color: var(--c-green);">✅ 答案：B. 关节僵硬</strong>

**踩分点一句话**：长时间石膏固定导致关节囊、韧带及周围软组织发生纤维性粘连和挛缩，造成关节活动受限，这是长期固定后最常见的并发症。

# 核心词扫盲

- **长腿石膏管型固定**：从大腿根部到足部的石膏固定方式，跨越膝关节和踝关节，目的是将骨折部位上下两个关节同时固定以稳定骨折端。本题中膝关节被固定4个月，导致关节囊挛缩。
- **关节僵硬**：指关节活动范围受限，但关节面本身仍完好，是由于关节囊、韧带、肌肉等软组织长时间不活动导致的纤维粘连和挛缩，是可逆的。
- **关节强直**：指关节面之间发生骨性或纤维性融合，导致关节完全丧失活动能力，是不可逆的改变。骨性强直提示关节面已破坏或融合，纤维性强直则可能是严重炎症的结果。

# 知识点精讲

## 关节僵硬——长期固定的必然代价

<span style="color: var(--c-blue);"><strong>关节僵硬</strong></span>是骨折外固定后最常见的并发症之一。其本质是：**关节固定不动 → 关节囊和韧带失去正常牵拉刺激 → 纤维组织增生、胶原纤维交联 → 关节囊挛缩、滑膜粘连 → 活动受限。**

关键在于病理层次：<span style="color: var(--c-blue);">关节面软骨是完好的</span>，问题出在<span style="color: var(--c-red);">关节周围软组织</span>。因此关节僵硬是可逆的——通过功能锻炼可以逐步恢复活动度。

## 与关节强直的本质区别

这是考试中反复考查的鉴别点：

|  | <span style="color: var(--c-blue);">关节僵硬</span> | <span style="color: var(--c-red);">关节强直</span> |
|------|------|------|
| 病变层次 | 关节囊、韧带、肌肉 | 关节面本身 |
| 病理改变 | 软组织挛缩、粘连 | 骨性融合或纤维性融合 |
| 可逆性 | 可逆，功能锻炼可恢复 | <strong style="color: var(--c-red);">不可逆</strong> |
| 常见原因 | 长期固定 | 化脓性关节炎、类风湿关节炎、关节内骨折 |

<span style="color: var(--c-yellow); background-color: var(--bg-yellow);">一句话鉴别：僵硬是“软组织锈住了”，强直是“骨头焊死了”。</span>

## 骨折愈合后功能障碍的常见原因排序

骨折患者拆除石膏后出现功能障碍，优先考虑 <strong style="color: var(--c-blue);">关节僵硬</strong>，这是最常见、最早出现、也最符合临床规律的原因。肌肉萎缩虽然也存在，但它导致的是肌力下降而非关节活动范围受限。骨折复位不理想或畸形愈合应该影响的是力线和肢体长度，而非单纯的膝关节活动度。

# 📋 考点速记

- <strong style="color: var(--c-green);">✅ 答案：B</strong>

- **口诀**：<span style="color: var(--c-blue); background-color: var(--bg-blue);">固定久了关节硬，早练早动防僵硬；钢板一拆活动差，软组织锈是根本。</span>

- **诊断公式**：<span style="color: var(--c-yellow); background-color: var(--bg-yellow);">骨折+长期石膏固定+拆石膏后关节活动受限=关节僵硬</span>

- **关键鉴别**：关节僵硬（软组织问题，可逆）vs. 关节强直（关节面问题，<strong style="color: var(--c-red);">不可逆</strong>）

- **同类题速记思维链**：有长期固定史→功能障碍→先考虑关节僵硬（最常见）→排除关节内病变后再考虑强直
