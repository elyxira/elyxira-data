### 📌 答案

<strong style="color: var(--c-green);">✅ 答案：C. 脐带过短</strong>

**踩分点一句话**：脐带过短在分娩时最常见的致命性问题是，胎先露下降时脐带被机械性拉紧、受压，导致血流中断，从而直接引发胎儿急性宫内窘迫。

### 核心词扫盲

*   **脐带先露 / 脐带脱垂**：指脐带位于胎先露部前方或一侧，甚至脱出宫颈口。这是急症，但它的直接危险是受压，而非题目问的几种异常中最易引起窘迫的机制。
*   **脐带假结**：因脐血管迂曲或华通胶局部堆积形成的形态学“结”，没有真结的收紧功能，通常**无临床意义**。
*   **脐带过短 / 过长**：正常脐带长度 <span style=\"color: var(--c-sky);\">30-70cm</span>。通常＜30cm为过短，＞70cm为过长。

### 知识点精讲

脐带的物理特性直接决定了分娩过程中胎儿的安全边界。

*   **核心物理机制**：脐带是连接胎儿与胎盘的“生命线”，必须在胎先露下降过程中始终留有足够的安全缓冲。当脐带**绝对或相对过短**时，产程中胎先露部的每一次强行下降，都会像“拽绳子”一样牵拉脐带。
*   **病理生理链条**：脐带被拉紧 → 脐血管受压变窄 → <strong style=\"color: var(--c-red);\">脐带血流急剧减少或中断</strong> → 胎儿-胎盘循环障碍 → <strong style=\"color: var(--c-red);\">急性胎儿缺氧</strong>（胎儿窘迫）。这种机械性梗阻可以在数分钟内危及胎儿生命。
*   **临床警示信号**：典型的表现为产程中胎心监护出现与宫缩同步的**晚期减速**或**重度变异减速**，甚至在胎先露衔接受阻时出现胎心异常。

### 📋 考点速记

<strong style=\"color: var(--c-green);\">✅ 答案：C. 脐带过短</strong>

*   **口诀**：“脐带短，拉得紧，血流断，胎儿窘。”
*   **诊断公式**：潜伏期/活跃期胎先露下降 + 胎心反复出现重度变异减速 → 警惕脐带机械性受压（过短/缠绕）。
*   **关键数字**：脐带长度 <span style=\"color: var(--c-sky); background-color: var(--bg-sky);\">正常范围：30-70cm</span>，<strong style=\"color: var(--c-red);\">＜30cm 为过短</strong>。
*   **病理生理链条**：<span style=\"color: var(--c-blue);\">脐带过短</span> → <span style=\"color: var(--c-red);\">胎头下降牵拉脐带</span> → <span style=\"color: var(--c-red);\">血管受压、血流中断</span> → <span style=\"color: var(--c-blue);\">急性胎儿窘迫</span>
