### 📌 答案

<strong style="color: var(--c-green);">✅ 答案: D. 孕满31周，宫底低于正常</strong>

踩分点: 从末次月经推算到就诊日恰好为 <strong style="color: var(--c-blue);">31周整</strong>；正常孕28周末宫底已达脐上三横指，此后每周上升约1cm，31周时宫底应远高于此水平，故当前高度显著偏低。

---

### 核心词扫盲

**宫底高度**: 指耻骨联合上缘至子宫底的垂直距离，是评估胎儿生长发育的床边筛查手段。正常妊娠不同孕周有对应参考范围——本题中"脐上三横指"约对应 <span style="color: var(--c-sky);">28周末</span> 的正常高度。

**末次月经**: 最后一次正常月经的第一天，是计算预产期和当前孕周的起点——方法: 末次月经第一天日期加280天即为预产期。

---

### 知识点精讲

**妊娠周数与宫底高度的对照关系**:

正常单胎妊娠，宫底高度随孕周有序上升:

| 孕周 | 宫底高度 |
|------|----------|
| 12周末 | 耻骨联合上2-3横指 |
| 16周末 | 脐耻之间 |
| 20周末 | 脐下1横指 |
| <strong style="color: var(--c-blue);">24周末</strong> | 脐上1横指 |
| <strong style="color: var(--c-blue);">28周末</strong> | 脐上3横指 |
| 32周末 | 脐与剑突之间 |
| 36周末 | 剑突下2横指 |
| 40周末 | 略下降至脐与剑突之间或稍高 |

这条表的核心规律: 20周宫底才出盆腔，28周正好在脐上三横指，此后 <span style="color: var(--c-blue); background-color: var(--bg-blue);">每增加1周，宫底约上升1cm（约1横指）</span>。

**宫底低于正常的意义**: 提示可能存在 <span style="color: var(--c-red);">胎儿生长受限</span>、<span style="color: var(--c-red);">羊水过少</span> 或孕周计算错误。本题中31周宫底停留在28周水平，低了两周以上，属于明显异常，需进一步 B 超评估。
