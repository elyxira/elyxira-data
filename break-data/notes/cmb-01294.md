<aside>两条呼吸链在这个点交汇——记住它，生化里考电子传递链这张图必用。</aside>

### 📌 答案

<strong style="color: var(--c-green);">✅ 答案：E. CoQ（辅酶Q）</strong>

**踩分点**：<span style="color: var(--c-blue);"><strong>NADH氧化呼吸链</strong></span>与<span style="color: var(--c-blue);"><strong>琥珀酸氧化呼吸链</strong></span>的唯一电子交汇点就是<span style="color: var(--c-yellow); background-color: var(--bg-yellow);"><strong>CoQ（辅酶Q）</strong></span>——两条链从不同上游递氢体出发，到CoQ汇合，再经同一段 cytochrome 路径传至O₂。

---

### 核心词扫盲

| 词条 | 解释 |
|------|------|
| <strong style="color: var(--c-blue);">电子传递链（呼吸链）</strong> | 位于线粒体内膜的一系列递氢体/递电子体，将NADH和FADH₂的电子最终传给O₂生成H₂O，同时驱动ATP合成 |
| <span style="color: var(--c-blue);"><strong>CoQ（辅酶Q/泛醌）</strong></span> | 脂溶性小分子，呼吸链中<span style="color: var(--c-yellow);">唯一非蛋白组分</span>，可自由穿行于线粒体内膜——正因如此，它成为两条链的交叉点 |
| <span style="color: var(--c-blue);">FMN</span> | 黄素单核苷酸，<strong>NADH氧化呼吸链</strong>的第一个电子受体，在CoQ上游 |
| <span style="color: var(--c-blue);">FAD</span> | 黄素腺嘌呤二核苷酸，<strong>琥珀酸氧化呼吸链</strong>的起始递氢体，同样在CoQ上游 |
| <span style="color: var(--c-blue);">cyt b / cyt c</span> | 细胞色素b和c，均位于CoQ<span style="color: var(--c-red);">下游</span>，两条链汇合后共同经过的路径，不是交叉点 |

---

### 知识点精讲

**两条呼吸链的完整路径**

<span style="color: var(--c-indigo);"><strong>NADH氧化呼吸链</strong></span>（主要途径）：
<span style="color: var(--c-blue);">NADH</span> → <span style="color: var(--c-blue);">FMN</span> → <span style="color: var(--c-yellow); background-color: var(--bg-yellow);"><strong>CoQ</strong></span> → <span style="color: var(--c-blue);">Cyt b</span> → Cyt c₁ → Cyt c → Cyt aa₃ → O₂

<span style="color: var(--c-indigo);"><strong>琥珀酸氧化呼吸链</strong></span>（FADH₂途径）：
<span style="color: var(--c-blue);">琥珀酸</span> → <span style="color: var(--c-blue);">FAD</span> → <span style="color: var(--c-yellow); background-color: var(--bg-yellow);"><strong>CoQ</strong></span> → Cyt b → Cyt c₁ → Cyt c → Cyt aa₃ → O₂

**结构逻辑**：CoQ之前，两条链各走各的路——一条靠FMN接电子，一条靠FAD接电子。CoQ及之后，<strong style="color: var(--c-blue);">共用同一段路径</strong>。这就是为什么CoQ被称为"交叉点"。

**命题常见陷阱**：
- 把交叉点写成<span style="color: var(--c-red);">cyt b</span>或<span style="color: var(--c-red);">cyt c</span>——这些是汇合后的共用段，不是分叉汇入点
- 把两条链的起点搞反——记死：<strong style="color: var(--c-blue);">NADH→FMN，FADH₂来自琥珀酸</strong>

### 📋 考点速记

<strong style="color: var(--c-green);">✅ 答案：E. CoQ</strong>

<span style="color: var(--c-yellow); background-color: var(--bg-yellow);"><strong>一句话口诀</strong></span>
> **两条呼吸链，汇在辅酶Q——FMN管NADH，FAD管琥珀酸**

<span style="color: var(--c-blue); background-color: var(--bg-blue);"><strong>同类题速记思维链</strong></span>
> ① 画出两条链的简化路径 → ② 找到上游分支各自独有的最后一个分子 → ③ 紧接着的共同分子就是交叉点

**易混淆对照**：

| | NADH呼吸链 | 琥珀酸(FADH₂)呼吸链 |
|------|------|------|
| 起始电子供体 | NADH | 琥珀酸 |
| 第一个递氢体 | <span style="color: var(--c-blue);">FMN</span> | <span style="color: var(--c-blue);">FAD</span> |
| 交叉点 | <span style="color: var(--c-yellow);"><strong>CoQ</strong></span> | <span style="color: var(--c-yellow);"><strong>CoQ</strong></span> |
| 交叉后路径 | Cyt b→c₁→c→aa₃→O₂ | 完全相同 |
