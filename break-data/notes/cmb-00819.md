### 核心词扫盲

**晚期减速（late deceleration）**
胎心监护中宫缩高峰期后出现的胎心率下降，恢复缓慢。提示<strong style="color: var(--c-red);">子宫胎盘灌注不足、胎儿缺氧酸中毒</strong>。频繁出现意味着胎儿窘迫已达危险程度，需尽快终止妊娠。

**ROA（右枕前位）**
胎头枕骨朝向母体骨盆右前方，是正常胎位之一，<span style="color: var(--c-sky);">属于可以阴道助产的胎位</span>。

**先露（station）**
胎头最低点与坐骨棘平面的关系。坐骨棘平面为"0"，以上为负值，以下为正值。<span style="color: var(--c-sky);">先露+3即胎头最低点已达坐骨棘下3cm</span>，是阴道助产的理想位置。

### 推理链条（压缩版）

1. **识别急迫性**：胎心100次/分（<span style="color: var(--c-red);">正常110-160</span>）+ 频繁晚期减速 = <strong style="color: var(--c-red);">胎儿窘迫，需尽快终止妊娠</strong>
2. **阴道助产硬条件**：宫口必须开全（10cm），先露必须足够低（≥+2，理想≥+3），骨盆出口无狭窄
3. **逐项筛查**：A宫口仅8cm→不合格；B先露-1（太高）且矢状缝未转正（持续枕后位）→不合格；D胎头在耻骨联合上方可触及较多部分→估计存在头盆不称→不合格；E骶骨凹度差+尾骨上翘→骨盆出口狭窄→不合格
4. **唯一合格项**：C满足宫口开全 + 先露+3 + 骨盆无异常，宫缩乏力恰好可以用产钳/胎吸克服

### 知识点精讲

<strong style="color: var(--c-blue);">阴道助产术（产钳/胎头负压吸引术）</strong>是在第二产程出现紧急情况时用来加速胎儿娩出的手术操作。核心原理是<strong style="color: var(--c-blue);">借助外力（牵引/负压）替代或增强母体产力</strong>，将胎儿从产道中快速牵出。

**必备条件（缺一不可）**：

| 条件 | 标准 |
|------|------|
| 宫口 | <strong style="color: var(--c-red);">开全（10cm）</strong> |
| 胎膜 | 已破 |
| 先露 | <strong style="color: var(--c-blue);">坐骨棘下≥+2，最佳+3或更低</strong> |
| 胎位 | 头先露，无头盆不称 |
| 骨盆 | 无明显狭窄，出口条件好 |
| 膀胱 | 排空 |

**适应证**包括：<span style="color: var(--c-red);">胎儿窘迫</span>、第二产程延长、母体因素（心脏病/高血压等需缩短第二产程）、宫缩乏力。

本题的关键陷阱在于：即使胎儿窘迫十万火急，只要阴道助产条件不满足，就必须改剖宫产——<strong style="color: var(--c-red);">没有"勉强试一下"的选项。</strong>

### 📋 考点速记

<strong style="color: var(--c-green);">✅ 答案：C</strong>

- **口诀**：<span style="color: var(--c-blue); background-color: var(--bg-blue);">助产三硬坎——宫口开全、先露够低、骨盆不窄；少一个就拉去剖</span>
- **诊断公式**：<span style="color: var(--c-yellow); background-color: var(--bg-yellow);">胎儿窘迫（晚期减速/胎心异常）+ 宫口开全 + 先露≥+3 + 骨盆正常 = 阴道助产指征</span>
- **绝对禁忌**：宫口未开全、先露-1或更高、明显头盆不称、骨盆出口狭窄（如骶骨凹度差、尾骨固定上翘）
- **关键数字**：宫口开全 = 10cm；先露合格线 ≥ +2，理想 ≥ +3；胎心正常范围 110-160次/分
- **同类题速记思维链**：① 先看宫口→未开全直接排除阴道助产 ② 再看先露→负值或+1不够低，排除 ③ 最后看骨盆→任何狭窄描述（骶骨/尾骨/坐骨棘异常）排除
