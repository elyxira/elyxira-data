### 📌 答案

<strong style="color: var(--c-green);">✅ 答案：D. 异长调节</strong>

**踩分点一句话**：搏出量减少是由于回心血量减少导致心室舒张末期容积减小、心肌初长度缩短，这是通过<span style="color: var(--c-blue);"><strong>异长调节（Frank-Starling机制）</strong></span>实现的自身调节，而非神经或后负荷因素。

### 核心词扫盲

- **异长调节**：即<span style="color: var(--c-blue);"><strong>Frank-Starling机制</strong></span>——心肌收缩力取决于心肌纤维的初长度。回心血量↑ → 心室舒张末期容积↑ → 心肌纤维拉长 → 收缩力↑ → 搏出量↑；反之亦然。这是心脏的<strong>自身调节</strong>，不依赖神经或体液。
- **等长调节**：在<strong>心肌初长度不变</strong>的情况下，通过改变心肌收缩能力（如交感神经兴奋增加Ca²⁺内流）来调节搏出量。与异长调节的核心区别在于"初长度是否改变"。
- **心室后负荷**：即<span style="color: var(--c-blue);">动脉血压</span>，心室射血时需克服的阻力。题干中动脉血压是<strong>降低</strong>的，后负荷并未增大。
- **心迷走神经**：释放<span style="color: var(--c-blue);">乙酰胆碱</span>，作用于M受体，使心率↓、房室传导↓、心房肌收缩力↓。对心室肌支配<strong>极少</strong>，不是调节心室搏出量的主要神经。

### 推理链条（压缩版，≤4步）

1. **体位改变 → 回心血量↓**：平卧→站立，重力作用使血液滞留于下肢静脉，回心血量减少。
2. **回心血量↓ → 心室舒张末期容积↓**：心脏泵出的血少了，充盈自然减少，心肌初长度缩短。
3. **初长度↓ → 心肌收缩力↓**：这是<span style="color: var(--c-blue);"><strong>异长调节</strong></span>的直接效应——心肌纤维缩短，粗细肌丝重叠状态偏离最适初长度，收缩力减弱。
4. **收缩力↓ → 搏出量↓ → 动脉血压↓**：在神经调节（交感/迷走）尚未介入的即刻阶段，搏出量的变化完全由异长调节解释。

### 知识点精讲

<span style="color: var(--c-indigo);"><strong>异长调节（Frank-Starling机制）</strong></span>是心肌最基本的自身调节，核心原理是：<strong>心肌收缩力与心肌纤维初长度成正比</strong>。心室充盈量越大，心肌拉得越长，粗细肌丝重叠越接近<strong>最适初长度（肌小节长度约2.2μm）</strong>，收缩力越强；反之，回心血量减少、充盈不足时，收缩力减弱。

这是心脏实现"<strong>进出平衡</strong>"的生理基础——右心回流量=左心输出量，防止血液在肺循环淤积。健康心脏在心率不变时，主要依赖异长调节来匹配回流量变化。

与<span style="color: var(--c-blue);"><strong>等长调节</strong></span>的鉴别是高频考点：异长调节是<strong>"调节初长度"</strong>（前负荷改变），等长调节是<strong>"调节收缩力本身"</strong>（Ca²⁺、交感兴奋等）。体位改变、输液、失血等引起前负荷变化时→异长调节；运动、应激等交感激活时→等长调节。

### 📋 考点速记

<strong style="color: var(--c-green);">✅ 答案：D</strong>

- **口诀**：<span style="background-color: var(--bg-yellow); color: var(--c-yellow);">回心血少初长缩，异长调节搏出弱</span>
- **诊断公式**：<span style="background-color: var(--bg-blue); color: var(--c-blue);">前负荷改变 + 搏出量变化 = 异长调节</span>
- **同类题速记思维链**：① 判断前负荷（回心血量/充盈压）是否改变 → ② 是→优先考虑异长调节；否→考虑等长调节或后负荷 → ③ 排除神经因素（除非题目明确问神经调节）
- **关键区分**：<strong>异长调节 = 初长度改变</strong> vs <strong>等长调节 = 收缩力本身改变</strong>
