<aside>呼吸商这道题，本质是分清"谁比谁"——永远是CO₂排放在分子，O₂摄入在分母。</aside>

### 📌 答案

<strong style="color: var(--c-green);">✅ 答案：B</strong>

<span style="color: var(--c-blue);"><strong>呼吸商</strong></span>定义为一定时间内机体呼出的<span style="color: var(--c-red);">CO₂量</span>与摄入的<span style="color: var(--c-sky);">O₂量</span>的比值，即 **CO₂/O₂**。B 之所以正确，是因为它准确描述了分子（呼出CO₂）和分母（摄入O₂）的关系。

### 核心词扫盲

- **呼吸商（RQ）**：反映机体利用何种营养物质供能的指标。正常值范围因混合饮食而波动于 <span style="color: var(--c-sky);">0.82-0.85</span> 之间。**RQ=1.0 提示糖供能，RQ=0.71 提示脂肪供能。**
- **摄入的 O₂ 与呼出的 CO₂**：机体氧化供能的化学计量表达。消耗的O₂是"输入"，产生的CO₂是"输出"——呼吸商是**产出/投入**的比值。

### 知识点精讲

<span style="color: var(--c-indigo);"><strong>呼吸商的核心生理意义</strong></span>

**呼吸商**是能量代谢研究中判断机体主要能量底物的关键指标。同一时间内，机体消耗了O₂产生CO₂，两者的摩尔比值揭示了当下主要的氧化供能物质。

<span style="color: var(--c-blue);"><strong>三大营养物质的呼吸商</strong></span>：
- **糖类**：RQ = <span style="color: var(--c-sky);">1.0</span>（完全氧化时CO₂产出等于O₂消耗）
- **脂肪**：RQ = <span style="color: var(--c-sky);">0.71</span>（脂肪分子中氧含量低，氧化时需消耗更多O₂，产CO₂相对少）
- **蛋白质**：RQ = <span style="color: var(--c-sky);">0.80</span>（介于两者之间）

**临床应用线索**：RQ 的变化可提示代谢底物的转换。如过度通气或代谢性酸中毒时，CO₂排出增多，RQ 可暂时升高。而在长期饥饿状态下，机体转向脂肪供能，RQ 可降至 0.7 附近。

### 📋 考点速记

<strong style="color: var(--c-green);">✅ 答案：B</strong>

- **口诀**：呼吸商，商出炉——CO₂出炉（呼出）做分子，O₂进口（摄入）做分母。
- **诊断公式**：`RQ = 呼出CO₂量 / 摄入O₂量`
- **关键数字**：
    - <span style="background-color: var(--bg-yellow);">糖类RQ = 1.0</span>
    - <span style="background-color: var(--bg-yellow);">脂肪RQ = 0.71</span>
    - <span style="background-color: var(--bg-yellow);">混合饮食RQ = 0.82-0.85</span>
- **同类题速记思维链**：呼吸商定义题 → 锁定分子（CO₂产出）与分母（O₂消耗）→ 排除单纯肺通气/肺容量的干扰项。
